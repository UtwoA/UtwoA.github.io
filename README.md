# utwoa portfolio site

Минималистичный адаптивный сайт-витрина (RU/EN) для GitHub Pages.

## Файлы

- `index.html` — разметка страницы
- `styles.css` — стили и адаптив
- `script.js` — переключение языка RU/EN

## Локальный запуск

Открой `index.html` в браузере.

## Деплой на GitHub Pages

1. Создай публичный репозиторий (например `utwoa.github.io` или любой другой).
2. Залей эти файлы в ветку `main`.
3. В GitHub открой `Settings` -> `Pages`.
4. В `Build and deployment` выбери:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` / `root`
5. Сохрани настройки и дождись публикации (обычно 1-3 минуты).

Если репозиторий называется `utwoa.github.io`, сайт будет доступен по адресу:
`https://utwoa.github.io`
